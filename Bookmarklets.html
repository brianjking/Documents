Bookmarklets

Simply drag these links to your browser's bookmarks bar and then click them while you're at the desired website.

Bypass Website Signup

These allow you to look for pre-created logins to websites via BugMeNot and Login2Me, using either the full domain name (like site.foo.com), or just the base domain name (foo.com). This is useful when you don't want to sign up to a site just to do some quick contribution. I suggest saving all four bookmarklets in a folder called "Bypass" or something, so that you have easy access to all four versions.

  • BugMeNot
  • Login2Me
  • BugMeNot (Domain Only)
  • Login2Me (Domain Only)

Automatic Per-site Mailbox Filler

Some people use catch-all mail domains, with separate mailbox names for each website they visit. Other people use a single mailbox, but with sub-addressing tags. This bookmarklet is for you. Simply click it, and it will automatically generate an email address for the website you are viewing and write it into the selected form-field (if possible, otherwise it displays it in a popup window). This allows you to effortlessly create per-website inboxes. For example, if you are on github.com, it would write "github-com@YOURDOMAIN.COM" into the form. You must edit the first value in this bookmarklet to change "YOURDOMAIN.com" to your own domain, or optionally change it to use sub-addressing instead. The "{}" symbol is where it inserts the current site's name, so to use sub-addressing with a "+" separator (the most common), you'd use a pattern like "YOURMAILBOXNAME+{}@YOURDOMAIN.com".
